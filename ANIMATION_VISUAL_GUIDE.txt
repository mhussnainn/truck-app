The "You Effect" is a modern animation pattern that makes elements feel interactive:

STEP 1: Scale Up (0.2s)          STEP 2: Glow Expansion (0.6s)
┌──────────────────┐             ┌──────────────────────────────┐
│  Normal Size     │             │ Small Box (start)            │
│  1x scale        │             │ box-shadow: 0 0 0 0px        │
│       ↓          │ Hover       │ ⟶ Expand outward ⟶          │
│  Larger          │  ────►      │ box-shadow: 0 0 0 30px       │
│  1.1x scale      │             │ (Glow expands like ripple)   │
└──────────────────┘             └──────────────────────────────┘

EFFECT: Makes the element feel like it's "reaching out" to the user

APPLIED ON 50+ ELEMENTS:
  ✓ All buttons (Call, Request, Submit)
  ✓ All interactive cards
  ✓ Navigation links
  ✓ Form inputs (on focus)
  ✓ Floating call button (extra large)

SCROLL REVERSIBILITY EXPLAINED
═══════════════════════════════════════════════════════════════════════════════════

NORMAL Animation:               REVERSIBLE Animation (with scrub):
┌──────────────────────────┐   ┌──────────────────────────────┐
│ Element at top of page   │   │ Element at top of page       │
│ ↓ Scroll down 500px      │   │ ↓ Scroll down 500px          │
│ Animation triggers once  │   │ Animation tied to scroll     │
│ ✗ Scroll back up?        │   │ position (smooth tracking)   │
│ Animation STAYS done     │   │ ✓ Scroll back up?            │
│ ✗ No reverse animation   │   │ Animation REVERSES smoothly  │
└──────────────────────────┘   │ ✓ Element returns to start   │
                               └──────────────────────────────┘

Implementation:
gsap.to(element, {
  y: 50,
  scrollTrigger: {
    trigger: element,
    scrub: 1,  ← Makes it reversible!
    start: 'top 80%',
    end: 'top 30%'
  }
});

Result: When you scroll the page back and forth, the animation follows
your scroll position smoothly - creating an immersive, responsive feel.
